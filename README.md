# О чем программа

Веб-интерфейс с бекендом, запускамым локально на компьютере, позволяет решать задачу дискретного логарифмирования на заготовленных эллиптических кривых над простым полем (их список может модифицироваться в `backend/api/curves.py`). Реализованы 3 метода:
- ро Полларда (универсальный алгоритм O(sqrt n))
- Похлига-Хеллмана (также O(sqrt n), но n - максимальный простой множитель в разложении порядка генератора)
- атака Смарта (для аномальных кривых с порядком генератора равным порядку поля, сложность сопоставимая с O(log n))

Структура `backend`:
- `api/` fastapi сервер для взаимодействия с решателями ecdlp
- `api/routes/` rest api эндпоинты (взаимодействие с решателями может осуществляться чисто с помощью rest api запросов на решение и опросом на статус задачи)
- `api/services/` модули для удобного создания задач в мультипоточном режиме (`TaskManager`) и работы с сокетами (`SocketManager`) - второй способ обращаться к солверам, этот способ используется в реализованном в директории `frontend` интерфейсе
- `curves/` независимая папка для работы с утилитой [ecgen](https://github.com/J08nY/ecgen), которую надо собрать и поместить в эту директорию для создания кривых
- `ecdlp/` сам модуль, предоставляющий интерфейс для программной работы с солверами
- `ecdlp/solvers/` реализации алгоритмов
- `ecdlp/utils/` вспомогательные утилиты для работы с математикой и тестовыми кривыми

# Запуск программы (frontend на vite+react+shadcn и backend на fastapi)

Потребуется `node >= 24`, python пакеты из `backend/requirements.txt`, запуск полностью автоматизирован с указанием порта. Потребуется также сборка vite проекта.

```
python run.py [-p 8000] [--port 8000]
```

# Добавление кривых

Как уже было отмечено, изменяя файл `backend/api/curves.py` можно менять кривые с которыми можно работать.

Рекомендуемый метод заключается в следующем.
1. Установить утилиту [ecgen](https://github.com/J08nY/ecgen), которую поместить в `backend/curves/`
2. `cd backend/curves`
3. Манипулировать с директориями `production/general/`, `production/smooth/`, `production/anomalous/` с помощью ecgen (пример - `generation_prod.sh`)
4. Скриптом `convert_to_api.py` автоматически согласовать кривые из `production/` с `backend/api/curves.py`
